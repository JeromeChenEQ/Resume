# Interactive Resume - Jerome Chen

A modern, interactive React web resume built with Vite and styled with CSS animations and responsive design.

## Features

✨ **Interactive Sections** - Collapsible resume sections for easy navigation
🎨 **Modern UI** - Clean, professional design with gradient backgrounds
📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile
⚡ **Fast Performance** - Built with Vite for instant development and optimized production builds
🔗 **Live Links** - Direct links to LinkedIn and GitHub profiles

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd c:\Projects\Resume
```

2. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The resume will automatically open in your browser at `http://localhost:5173`

### Build for Production

Create an optimized production build:
```bash
npm run build
```

The built files will be in the `dist/` directory.

### Preview Production Build

Preview the production build locally:
```bash
npm run preview
```

## Content

The resume includes:
- **Contact Information** - Email, phone, LinkedIn, and GitHub
- **Education** - Singapore Management University details
- **Experience** - Teaching Assistant and Deputy Director roles
- **Projects** - WADE and Spectrum projects
- **Skills** - Technical skills, soft skills, and languages

## Interactive Features

- **Collapsible Sections** - Click section headers to expand/collapse content
- **Smooth Animations** - Content fades in with smooth transitions
- **Hover Effects** - Interactive elements respond to mouse hover
- **Responsive Design** - Automatically adapts to different screen sizes
- **External Links** - Direct navigation to LinkedIn and GitHub profiles
